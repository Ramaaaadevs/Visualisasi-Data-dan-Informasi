## 📌 Tentang Project

Project ini merupakan tugas mata kuliah **Visualisasi Data dan Informasi** yang menganalisis data kecelakaan lalu lintas di UK. Visualisasi mencakup:

- Peta sebaran lokasi kecelakaan berdasarkan tingkat keparahan
- Top wilayah dengan frekuensi kecelakaan tertinggi
- Tren kecelakaan berdasarkan waktu dan kondisi jalan
